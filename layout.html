CSS Layouts

Flexbox

Flexbox, også kalt flex, brukes til å gjøre CSS på elementer i én dimensjon. Hvis man har en rekke elementer så er det god idé å bruke flex. Med flex kan man midtjustere, endre rekkefølge, lage mellomrom, og sikkert mye mer. Retningen til flexboxen er vannrett som default, men du kan få den til å bli loddrett ved å sette flex-direction til column. Utover det finnes det mange måter å stilsette flexboxen, som for eksempel å midtjustere elementene inni.

eksempel:

#container{
	display: flex;
	flex-direction: column;
	justify-content: center;
	align-items: center;
}

Grid

Et element med display: grid får underelementene sine plassert i et rutenett. Noe som er veldig nyttig med grid er grid-template-areas, hvor man kan bestemme hvor elementene skal havne i rutenettet med en rekke string-verdier. Eksempelet under viser hvordan det kan gjøres.

eksempel:

#container{
	display: grid;
	grid-template-areas:
		"menu menu menu"
		"main main sidebar"
		"footer footer footer";
	grid-template-columns: repeat(3, 1fr);
	grid-template-rows: 1fr 3fr 1fr;
}
#menu{
	grid-area: menu;
}
#main{
	grid-area: main;
}
#sidebar{
	grid-area: footer;
}
#footer{
	grid-area: footer;
}

Float

Float brukes til å flytte elementer til høyre eller venstre. Float kan være vanskelig å bruke, fordi elemented blir tatt ut av foreldreelementet. Det kan føre til at foreldreelementet blir tomt og kollapser. Det finnes måter å løse det problemet, men for mange er det ikke værdt bryet. Derfor er det som oftest bedre å bruke flex eller grid. På dette nettstedet brukes float på noen av bildene fordi jeg liker hvordan teksten legger seg rundt bildene.
